create a new file in order to debug git tools
now we need to first try to command 'touch' create a new file
second try to command 'vi' <file> edit current file 
esc :wq 
git add <file>  本文件添加到本地仓库
git status view current file's status
git commit -m "备注"   提交到本地库
git diff <file> 对比版本库和当前修改后的不同之处

git push 提交到远程仓库
